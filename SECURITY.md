# Security Policy

## Security Dataset for AI Training

This repository contains **SecureCode v2.0**, a production-grade dataset of security-focused coding examples for training security-aware AI models.

### Dataset Purpose
- ✅ Educational security training data
- ✅ AI model fine-tuning for secure code generation
- ✅ Research on LLM security capabilities
- ✅ OWASP Top 10 vulnerability coverage

**Important:** This is defensive educational data with examples of both vulnerable and secure code for training purposes.
